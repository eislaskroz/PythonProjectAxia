MODIFICACIONES - LEVANTAMIENTO SEGURIDAD Y MONITOREO

1. Infraestructura existente/reutilizable:
- Al seleccionar "No" en "¿Existe infraestructura para Seguridad y Monitoreo?", se deshabilitan:
  - Tipo de infraestructura existente
  - Estado general
  - Observaciones de infraestructura
- Al seleccionar "Sí" o "Parcial", los campos vuelven a habilitarse.

2. Rack, gabinete y energía:
- Las preguntas se cambiaron a formato específico:
  - ¿Se requiere rack?
  - ¿Se requiere gabinete?
  - ¿Se requiere UPS?
  - ¿Se requiere contacto regulado?
  - ¿Se requiere tierra física?
- Si la respuesta es "Sí", se habilita un campo de detalle/tipo para cada concepto.
- Si la respuesta es "No", el campo de detalle queda deshabilitado y se limpia.

3. Datos técnicos Seguridad y Monitoreo:
- Se actualizaron las leyendas a preguntas más claras:
  - ¿Cuántas cámaras se requieren?
  - ¿Qué tipo de cámaras se requieren?
  - ¿Dónde se ubicará el NVR/DVR?
  - ¿Dónde estará el punto de enlace de red?
  - ¿Dónde estará el punto de energía?
  - ¿Cuántos días de trabajo se proyectan?
  - ¿Cuántas personas se consideran?

4. Descripción del levantamiento:
- Para Seguridad y Monitoreo se elimina visualmente la sección "Descripción del levantamiento".
- Para no romper el flujo de guardado, el sistema genera internamente la descripción con el resumen técnico del levantamiento.
- En otros tipos de levantamiento se conserva la descripción.

5. Guardado/PDF:
- Se agregan los detalles nuevos dentro del JSON técnico y resumen PDF.

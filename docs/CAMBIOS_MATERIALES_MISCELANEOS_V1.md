# Materiales misceláneos V1

Se agregó una sección común a todos los levantamientos con materiales sugeridos por especialidad.

- Cada material inicia en **No**.
- Al seleccionar **Sí**, se habilitan cantidad, unidad y especificación.
- Se pueden agregar materiales adicionales.
- En levantamientos se guardan dentro de `lev_detalle_tecnico_json.materiales_miscelaneos`.
- En Obra Civil se guardan dentro de `obc_ejecucion_json.materiales_miscelaneos`.
- Los materiales se incorporan al resumen/PDF para servir como base de cotizaciones y órdenes de trabajo.
